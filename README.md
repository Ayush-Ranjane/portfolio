# Ayush Ranjane Portfolio

A personal portfolio website for Ayush Ranjane, focused on AI, machine learning, predictive analytics, time-series forecasting, and environmental intelligence.

## Overview

This project is a single-page portfolio built with HTML, Tailwind CSS via CDN, and vanilla JavaScript. It includes:

- A hero section and personal introduction
- About and skills sections
- Project showcase with modal details
- Resume access
- Contact form with mailto-based submission

## Getting Started

Since this is a static website, you can run it locally by opening `index.html` in your browser.

For a smoother development experience, you can also serve it with a simple local server, for example:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Customization

- Update content directly in `index.html`
- Replace images in the `images/` folder
- Replace the resume in `doc/resume.pdf`
- Update project links, profile links, and contact details in the page source

## Deployment

This portfolio can be deployed easily to static hosting providers such as:

- Vercel
- Netlify
- GitHub Pages
